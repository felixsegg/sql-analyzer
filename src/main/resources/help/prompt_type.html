Prompt Type Details Help

This window allows you to create and edit prompt types used for generating LLM prompts.
  • Name:
    Enter a short and descriptive name for the prompt type. This will be shown in selection lists.
  • Description:
    Optionally provide a brief explanation of when this type should be used.
  • Usage:
    Prompt types are selected when creating a Prompt. They categorize prompt phrasing styles (e.g., descriptive, technical, goal-oriented, keyword) and help organize filtering and analysis.
