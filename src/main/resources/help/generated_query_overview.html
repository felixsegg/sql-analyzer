Generated Query Overview Help

This window displays a list of all SQL queries that have been generated by LLMs in response to prompts.
  • View details:
    Double-click any entry, press Enter, or use the context menu ("Show details") to view the properties of a generated query.
  • Add new generated query:
    Click the "Add new..." button to manually create a generated query (not recommended in typical workflows).
  • Delete generated query:
    Select a generated query and click "Delete", or press the Delete key to remove it from the list.
  • Refresh list:
    Click "Refresh" to reload the list, for example after changes were made elsewhere.
Note: Generated queries are usually created automatically. Manual creation is mainly for exceptional cases or testing.
