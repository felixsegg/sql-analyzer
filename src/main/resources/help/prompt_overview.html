Prompt Overview Help

This window displays a list of all prompts used in the evaluation process.
  • View details:
    Double-click any entry, press Enter, or use the context menu ("Show details") to view or edit a prompt.
  • Add new prompt:
    Click the "Add new..." button to create a new prompt.
  • Delete prompt:
    Select a prompt and click "Delete", or press the Delete key to remove it from the list.
  • Refresh list:
    Click "Refresh" to reload the list, for example after changes were made elsewhere.
