LLM Details Help

This window allows you to define and manage Large Language Models (LLMs) used by the SQL evaluation tool.
  • Name:
    Provide a descriptive name for the LLM. This helps you identify it in lists.
  • API:
    Choose the provider used to communicate with the LLM. You can show/hide dummy providers using the checkbox.
    Available options include:
    • OpenAI (ChatGPT)
    • Google Gemini
    • Anthropic Claude
    • DeepSeek AI
    • Numerical dummy (returns random numbers; no API key; no real rate limit)
    • SQL dummy (returns a fixed SQL query; no API key; no real rate limit)
    • Rate-limited numerical dummy (simulates ~5% rate-limit failures)
    • Rate-limited SQL dummy (simulates ~5% rate-limit failures)
  • API key:
    Enter the API key required by the selected provider. Not needed for dummy providers.
  • Model:
    Specify the model identifier as required by the provider (e.g., gpt-4o-mini-2024-07-18, claude-sonnet-4-20250514). Not required for dummy providers.
  • Min and Max Temperature:
    Define a small range used across repetitions to avoid cached responses from providers. This is not intended to test creativity.
