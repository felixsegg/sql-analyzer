Generated Query Details Help

This window shows a single SQL statement that was automatically generated by an LLM from a specific prompt. These entries are later used in the evaluation and CSV export.
  • Prompt:
    The prompt this query was generated from (belongs to a sample query and a prompt type).
  • LLM:
    The language model that produced this SQL.
  • SQL:
    The generated SQL statement.
  • Origin & repetitions:
    Multiple generated queries may exist for the same Promptâ€“LLM pair (e.g., due to repetitions and the temperature range used to avoid cached responses).
  • Workflow:
    Generated queries are evaluated against their sample query in the Evaluation window; results can be exported as CSV for further analysis (e.g., in R).

Note: Generated queries are intended to be system-created and kept unchanged. Manual editing is not recommended; if you modify a query, re-run the evaluation to keep results consistent. You can delete a generated query if you want to exclude it from evaluation.
