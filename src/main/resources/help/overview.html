Overview Help

This window shows an overview of a set of domain objects used in the SQL evaluation tool.
  • View details:
    Double-click any entry, press Enter, or use the context menu (“Show details”) to view or edit an object.
  • Add new:
    Click the “Add new…” button to create a new object.
  • Delete:
    Select an object and click “Delete”, or press the Delete key to remove it.
  • Filter:
    Use the filter panel to narrow down the list.
    • Text filters search case-insensitively across the chosen string field.
    • Dropdown filters let you select a specific value; the count in parentheses shows how many objects have that value in total.
    • An external filter may be applied automatically; remove it via the “Remove external filter” button.
    • Multiple filters are combined using AND.
  • Sorting:
    Items are listed in alphabetical order.

Tip: Right-click an entry to open the context menu.
