Overview Help

This window shows an overview of a set of domain objects used in the SQL evaluation tool.
  • View details:
    Double-click any entry, press Enter, or use the context menu ("Show details") to view or edit the properties of an object.
  • Add new:
    Click the "Add new..." button to create a new object.
  • Delete:
    Select an object and click "Delete", or press the Delete key to remove it from the list.
  • Filter:
    Use the filter panel to narrow down the list of objects.
    • Text filters allow you to search by name or other string properties (case-insensitive).
    • Dropdown filters let you select specific attribute values; counts in parentheses show how many objects share that value in total.
    • An external filter may be applied automatically; you can remove it with the provided button.
    • Multiple filters will be combined conjunctivally.
