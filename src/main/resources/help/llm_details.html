LLM Details Help

This window allows you to define and manage Large Language Models (LLMs) used by the SQL evaluation tool.
  • Name:
    Provide a descriptive name for the LLM. This name will help you identify the model within the application.
  • API:
    Choose the API provider used to communicate with the LLM.
    Available options are:
    • OpenAI (ChatGPT)
    • Google Gemini
    • Anthropic Claude
    • DeepSeek AI
    • Numerical dummy (returns random numbers for demo/testing)
    • SQL dummy (returns a fixed SQL query for demo/testing)
  • Model:
    Specify the model identifier as required by the selected API.
    Examples: gpt-4o-mini-2024-07-18, claude-sonnet-4-20250514
  • Min and Max Temperature:
    Set the range for the temperature parameter. Lower values (close to 0) produce more predictable output. Higher values (up to 1) produce more varied and creative responses.

    Recommendation: Choose a small range (e.g., 0.7-0.8) to prevent cached responses and encourage diverse results.
