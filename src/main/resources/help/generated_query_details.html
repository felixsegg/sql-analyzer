Generated Query Details Help

This window displays details of SQL queries that have been generated by an LLM based on a specific prompt.
  • Prompt:
    Shows the prompt on which this query was based.
  • LLM:
    Indicates the language model that generated this query.
  • SQL:
    Displays the SQL statement generated by the LLM.
Note: Generated queries are typically created automatically by the system. Manual creation or editing is not recommended.
