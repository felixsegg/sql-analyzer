Sample Query Overview Help

This window shows a list of all configured sample queries used in the evaluation process.
  • View details:
    Double-click any entry, press Enter, or use the context menu ("Show details") to view or edit a sample query.
  • Add new sample query:
    Click the "Add new..." button to create a new sample query.
  • Delete sample query:
    Select a sample query and click "Delete", or press the Delete key to remove it from the list.
  • Refresh list:
    Click "Refresh" to reload the list, for example after changes were made elsewhere.
