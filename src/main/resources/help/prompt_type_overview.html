Prompt Type Overview Help

This window displays a list of all available prompt types used for LLM prompt generation.
  • View details:
    Double-click any entry, press Enter, or use the context menu ("Show details") to view or edit a prompt type.
  • Add new prompt type:
    Click the "Add new..." button to create a new prompt type.
  • Delete prompt type:
    Select a prompt type and click "Delete", or press the Delete key to remove it from the list.
  • Refresh list:
    Click "Refresh" to reload the list, for example after changes were made elsewhere.
