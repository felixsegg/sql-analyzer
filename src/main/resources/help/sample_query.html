Sample Query Details Help

This window allows you to create and edit sample SQL queries for use in LLM evaluation and prompt generation.
  • Name:
    Enter a concise and descriptive name for the sample query.
  • Description:
    Optionally add further context or notes about the query and its purpose.
  • Complexity:
    Select the query’s complexity level. Possible values:
    • Low: Simple query, few joins, no subqueries or aggregations.
    • Mid: Multiple joins, simple aggregations, simple subqueries.
    • High: Multiple joins, complex subqueries, advanced filter logic, or aggregations over several tables.
  • Prompt Context:
    Define a text template used to generate prompts for LLMs. This must include the placeholder §§§, which will be replaced by the specific question or task. The placeholder must appear exactly once.
  • SQL:
    Enter the reference SQL statement for this sample query.
    The statement must start with SELECT.
