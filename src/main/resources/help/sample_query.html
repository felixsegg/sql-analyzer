Sample Query Details Help

This window allows you to create and edit sample SQL queries for use in LLM evaluation and prompt generation.
  • Name:
    Enter a concise and descriptive name for the sample query.
  • Description:
    Optionally add context or notes about the query and its purpose.
  • Complexity:
    Select the query’s complexity level. This is used for classification, filtering, and analysis; it does not change generation behavior.
    Possible values:
    • Low: Simple query, few joins, no subqueries or aggregations.
    • Mid: Multiple joins, simple aggregations, simple subqueries.
    • High: Multiple joins, complex subqueries, advanced filter de.seggebaeing.sqlanalyzer.logic, or aggregations over several tables.
  • Prompt Context:
    Define a text template used to build the LLM prompt. The placeholder §§§ will be replaced at runtime by the specific prompt text, and must appear exactly once.
    You can include brief schema or domain hints here to guide the model.
  • SQL:
    Enter the reference SQL statement for this sample query. The statement must start with SELECT.
