Prompt Details Help

This window allows you to create and edit prompts that are used for LLM evaluations.
  • Sample query:
    Select the sample query this prompt belongs to.
  • Prompt type:
    Choose the prompt type, which defines the phrasing style (e.g., descriptive, technical, goal-oriented, keyword).
  • Prompt text:
    Enter the variable part of the request. Keep it aligned with the chosen type; schema or domain hints belong in the sample query’s prompt context.
  • How it’s used:
    During generation, the tool builds the full model prompt by inserting this text into the selected sample query’s prompt context at the placeholder §§§. The same prompt can be used with multiple LLMs and across repetitions.
