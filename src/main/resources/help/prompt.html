Prompt Details Help

This window allows you to create and edit prompts that are used for LLM evaluations.
  • Sample query:
    Select the sample query that this prompt is based on.
  • Prompt type:
    Choose the prompt type, which defines the style or approach for formulating the prompt.
  • Prompt text:
    Enter the actual text of the prompt. The content should match the requirements for the selected prompt type and sample query.
