LLM Overview Help

This window shows a list of all configured Large Language Models (LLMs) used in the SQL evaluation tool.
  • View details:
    Double-click any entry, press Enter, or use the context menu ("Show details") to view or edit the properties of an LLM.
  • Add new LLM:
    Click the "Add new..." button to create a new LLM configuration.
  • Delete LLM:
    Select an LLM and click "Delete", or press the Delete key to remove it from the list.
  • Refresh list:
    Click "Refresh" to reload the list of LLMs, for example after changes were made elsewhere.
